# Seguimiento de Problemas (Issues)

Este directorio contiene los problemas abiertos que debe atender el agente AFK.
Cada problema debe ser un archivo de Markdown (`.md`) con una descripción clara de la tarea o bug.

## Flujo:
1. Crea un archivo en esta carpeta, ej: `fix_login_bug.md`.
2. El agente lo leerá y, si es prioritario, intentará resolverlo.
3. Una vez completado, el archivo se debe mover a `issues/done/`.
